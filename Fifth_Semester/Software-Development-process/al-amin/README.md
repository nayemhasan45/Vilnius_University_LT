# My Data Visualization Projects

This repository contains two data visualization projects that use open data APIs to present information in an interactive way.

---

## 1. World Bank Data Comparison: Bangladesh vs. Lithuania

This project fetches, processes, and visualizes population and education data from the World Bank to compare trends between Bangladesh and Lithuania. It also includes mock district-level data for a more granular comparison.

*   **Technologies:** Python, Pandas, React, Recharts, Tailwind CSS
*   **Live Demo:** [https://flourishing-bavarois-ef6168.netlify.app/](https://flourishing-bavarois-ef6168.netlify.app/)

---

## 2. Eurostat Unemployment Data Visualization

This application visualizes the yearly average unemployment rate in Lithuania from 2015 to 2024. The data is fetched directly from the Eurostat API and displayed in an interactive chart.

*   **Technologies:** React, Recharts, Vite
*   **Live Demo:** [https://splendid-crepe-c507dc.netlify.app/](https://splendid-crepe-c507dc.netlify.app/)
